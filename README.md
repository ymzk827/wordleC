# Wordle in C

A simple console-based remake of "Wordle game" written in C. 
I made this cause i was bored and deside to practice in programming with C.

## Features

- Console gameplay like Wordle
- Random word selection from 300+ words from file `words.txt` (add your words if you want)
- 5-letter word validation
- Feedback on correctly guessed letters
